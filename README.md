## 🚀 Cài đặt & Chạy dự án

```bash
# Sao chép repo
git clone https://github.com/NekoTheDev/New-Port.git
cd New-Port

# Cài dependencies
npm install

# Chạy local
npm run dev

# Build production
npm run build
```
